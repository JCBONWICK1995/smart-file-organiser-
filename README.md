# smart-file-organiser-
A python utility that automatically sorts files into catagorized folders based on their extractions.

## features
- Automatically detects file types (Images, Docs, Videos, etc.)
- Creates destination folders dynamically.
- Handles unknown file types by moving them to an 'Others' folder.

- ##  How to run
- 1. Clone this repo.
  2. Run 'python main.py'.
  3. Provide the path to the folder you want to organize.
